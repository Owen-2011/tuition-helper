# 学费助手 - 独立网站版

一个完全独立的课外机构学费管理工具，可以在任何地方、任何时间使用，无需依赖任何服务器或网络连接。

## 特性

✅ **完全独立** - 单个 HTML 文件，无需后端服务器  
✅ **离线可用** - 所有数据存储在浏览器本地  
✅ **无需登录** - 直接打开即可使用  
✅ **自动计算** - 自动计算应收学费和缺勤减免  
✅ **数据持久化** - 刷新页面后数据不丢失  
✅ **响应式设计** - 支持桌面、平板和手机  

## 快速开始

### 方式一：直接打开（最简单）

1. 下载 `index.html` 文件到本地
2. 用浏览器打开 `index.html` 文件
3. 立即开始使用！

### 方式二：部署到服务器

如果您想在线部署，可以上传到任何支持静态文件的服务器：

- **免费选项**：GitHub Pages、Netlify、Vercel 等
- **付费选项**：任何虚拟主机或云服务器

## 使用说明

### 添加学生

1. 点击右下角的 **+** 按钮
2. 填写学生信息：
   - 学生姓名（必填）
   - 课程名称（必填）
   - 总课时数（必填）
   - 单价/课时（必填）
   - 缺勤次数（可选）
3. 点击"添加"保存

### 查看学生详情

- 点击学生卡片查看完整信息
- 可以编辑或删除学生记录

### 搜索学生

- 使用顶部搜索框按名字或课程名搜索
- 支持模糊搜索

### 查看统计

- 切换到"📊 统计"标签页
- 查看总学生数、待结算金额等数据

### 标记结算

- 在学生详情页面标记为"已结算"
- 统计页面会自动更新

## 学费计算

应收学费 = 总课时 × 单价 - 缺勤减免

**缺勤减免** = 缺勤次数 × 单价

例如：
- 学生 A：10 课时 × ¥50/课时 = ¥500（无缺勤）
- 学生 B：10 课时 × ¥50/课时 - 2 次缺勤 × ¥50 = ¥400

## 数据存储

所有数据存储在浏览器的 **localStorage** 中：

- ✅ 数据完全保存在本地设备
- ✅ 不会上传到任何服务器
- ✅ 隐私完全保护
- ✅ 刷新页面后数据不丢失

### 备份数据

如果需要备份，可以：

1. 打开浏览器开发者工具（F12）
2. 进入 Console 标签
3. 执行：`copy(localStorage.getItem('tuition_manager_students'))`
4. 粘贴到文本文件保存

### 恢复数据

1. 打开浏览器开发者工具（F12）
2. 进入 Console 标签
3. 执行：`localStorage.setItem('tuition_manager_students', '粘贴之前复制的内容')`
4. 刷新页面

## 常见问题

**Q: 数据会丢失吗？**  
A: 不会。数据存储在浏览器本地，除非您清除浏览器缓存或卸载浏览器。

**Q: 可以在多个设备上使用吗？**  
A: 可以。每个设备有独立的数据存储。如需同步，可以手动备份和恢复。

**Q: 需要网络连接吗？**  
A: 不需要。完全离线可用。

**Q: 可以导出数据吗？**  
A: 可以。参考上面的"备份数据"部分。

## 部署到免费服务

### GitHub Pages

1. 创建 GitHub 账户
2. 新建仓库 `tuition-helper`
3. 上传 `index.html`
4. 在 Settings → Pages 中启用 GitHub Pages
5. 访问 `https://你的用户名.github.io/tuition-helper`

### Netlify

1. 访问 [netlify.com](https://netlify.com)
2. 拖拽 `index.html` 上传
3. 自动生成公开链接

### Vercel

1. 访问 [vercel.com](https://vercel.com)
2. 导入项目或拖拽上传
3. 自动部署
